Use these tags to start your first website! These tags can be used to make a basic, functional website!

<> ----------- The information between these brackets tells your website to do something! Make sure every piece of code that you have is betweek these tags!

</> ---------- Closes out a tag and ends the command! Make sure to use this everytime you tell your website to do something!

<!DOCTYPE HTML> ---------- Informs notepad that we are working in an HTML document!

<html> ---------- Informs notepad that we are about to start typing in html

</html> ---------Informs notepad that we are done typing in html

<font> Any text that appears between the font tags will appear as text in your website </font>

<head> Information that appears between these tags will not appear in your website </head>

<head>
<title> This gives your website an official title. This will appear in the top of your browser as the name of the tab.</title>
<head>

<br> ------- Sends your text to the next line. You do not need to close this!

<hr> ------- Sends your text to the next line and adds a horizontal line accross your full site kindof like what you see below this!



See below for an example of how to use all of these together! Feel free to copy and paste into your own notepad document to test it out!
