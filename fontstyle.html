Use these to edit the look of the text in your website.




<center> This centers the text listed between your tags on your website. </center>


<b>bolded text</b>
<i>italicized text</i>
<del>deleted text</del>
<mark>highlighted text</mark>

The following edits all belong within the <font> tag. Make sure to see the final instruction below on what to do, and what not to do with these!


<font style="insert style name here"> ---------Changes the font of your text. Check out This link for a list of font families.

<font style=”font-size:number%;”>----------- Changes the size of the text

<font style="color=color name"> --------- Changes the color of the text. See the following link for color codes: HTML Color codes

You can group each of these together in one font tag. This is much cleaner than a new tag for each change to your text. Here is an example:

<font style=”font-size:300%; font-family:courier; color:blue”>
